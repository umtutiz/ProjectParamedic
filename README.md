# 🚑 Project Paramedic [WIP]

> **Note:** This project is currently a Work In Progress (WIP). Active development and refactoring are ongoing.

## 📖 About The Project
Project Paramedic is an atmospheric, multiplayer-focused game developed in Unity. The core vision revolves around cooperative gameplay mechanics where players navigate challenging scenarios. 

This repository serves as a showcase of my active learning and implementation of multiplayer networking architectures and game development patterns.

## ✨ Key Features (Currently Implemented)
* **Multiplayer Lobby System:** Integrated **Facepunch.Steamworks** for seamless Steam-based matchmaking and lobby creation.
* **Network Relay System:** Custom relay architecture to handle player connections and data synchronization across the network.
* **Objective Tracking:** Built-in objective indicator system to guide players through dynamic scenarios.
* **UI/UX Menus:** Main menu and lobby interface integration.

## 🛠️ Built With
* **Game Engine:** Unity (C#)
* **Networking & API:** Facepunch Steamworks (`steam_api64`)
* **Version Control:** Git & GitHub

## 🚀 Getting Started
*To run this project locally, ensure you have the Steam Client running in the background, as the Facepunch API requires an active Steam connection to initialize the `steam_appid`.*
